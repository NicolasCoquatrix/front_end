Bloctel.fr

Année 2021

Développement Web

Je suis fier de vous présenter mon projet phare : Bloctel.fr, un site web dédié à la protection contre le démarchage téléphonique. À travers ce projet, j'ai mis en œuvre mes compétences en développement web pour offrir une solution efficace et conviviale à tous les utilisateurs.

Informations

Bloctel.fr est une plateforme conçue pour permettre aux utilisateurs de s'inscrire facilement et gratuitement sur la liste d'opposition au démarchage téléphonique en France. Grâce à cette inscription, les utilisateurs bénéficient d'une protection renforcée contre les appels indésirables. De plus, le site offre la possibilité de signaler les appels persistants ou abusifs afin de prendre des mesures appropriées.

Technologies
  • Front-end : HTML5, CSS3, JavaScript
  • Back-end : PHP, MySQL
  • Framework : Laravel
  • CMS : WordPress

Adopteunmec.com

Développement Web

+

Adopteunmec.com

Année 2023

Développement Web

J'ai eu le plaisir de développer le site web AdopteUnMec.com, une plateforme de rencontre en ligne qui propose une approche unique et amusante pour trouver l'amour. À travers ce projet, j'ai mis en œuvre mes compétences en développement web pour créer une expérience de rencontre en ligne unique en son genre.

Informations

AdopteUnMec.com est bien plus qu'un simple site de rencontre. Il propose une approche originale où ce sont les femmes qui ont le pouvoir de choisir et d'initier les conversations. Les hommes, quant à eux, sont présentés sous forme de produits dans un "catalogue" virtuel. Cette approche ludique et décalée offre une expérience rafraîchissante aux utilisateurs, en mettant l'accent sur le respect, l'authenticité et la possibilité de trouver une connexion réelle.

Technologies
  • Front-end : HTML5, CSS3, JavaScript
  • Framework : Ruby on Rails
  • Base de données : MySQL
  • Système de paiement : Stripe
  • Outils de conception graphique : Adobe Photoshop, Adobe Illustrator
